HTML
(linguagem de marcação de hipertexto) é o bloco de construção da web. Define o significado e a estrutura do conteudo da web Outras tecnologias do HTML geralmente são usadas para descrever a aparência/apresentação(CSS) ou a funcionalidade/comportamento (javascript) de uma página da web.
